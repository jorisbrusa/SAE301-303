Nauralytics : L'Analyse au Service de la Nature

Description du projet

Dans les champs, sous des serres ou en plein air, mais aussi dans les parcs et les jardins, on peut utiliser des méthodes différentes pour protéger les plantes contre les maladies, les insectes, les virus, les bactéries, les acariens ou encore les mauvaises herbes. Ces méthodes, appelées “biocontrôle”, utilisent les mécanismes naturels et les interactions qui existent déjà dans la nature pour résoudre ces problèmes.
